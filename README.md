# Fuel Tracker Web App

A modern, responsive web application built with HTML, CSS, and Vanilla JavaScript to easily verify whether a vehicle is eligible to pump fuel based on its license plate number and the selected date.

## Features

- **Eligibility Check:** Input the last digit of your license plate and select a date to determine your fueling eligibility.
- **Multi-language Support:** Use the application in English, Sinhala, or Tamil.
- **Glassmorphism UI:** Stunning, modern dark theme design with glassy UI elements.
- **History Log:** Check your recent history to review past eligibility lookups.
- **Fully Responsive:** Works seamlessly on mobile devices, tablets, and desktops.

## Quick Start

You can access the live version of the project hosted on GitHub Pages here: [Fuel Tracker App](https://bawan2001.github.io/Fuel_Tracker/)

Alternatively, to run the application locally:
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Bawan2001/Fuel_Tracker.git
   ```
2. Navigate into the project folder and open `index.html` in your web browser.

## Technologies Used

- **HTML5:** For the basic structure and forms.
- **CSS3:** For styling, custom animations, and a responsive layout using custom utility classes prioritizing Glassmorphism aesthetics.
- **Vanilla JavaScript:** For logic, translation processing, and history tracking. No external libraries are used!
